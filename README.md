## 📌 项目简介

本项目面向**工厂、实验室、仓库**等场景中的**药品搬运与分类回收**需求，旨在解决人工操作中**气体泄漏、化学品灼伤**等安全隐患。系统以 **RDK X5** 为核心算力单元，**STM32H743** 为下位机执行控制器，构建了一套全自主的 **“感知—决策—执行”** 任务闭环。

## 🎯 应用场景

| 场景 | 典型任务 |
|:---|:---|
| 🏭 工厂 | 危险化学品搬运、分类回收 |
| 🧪 实验室 | 实验药品取用、废液回收 |
| 📦 仓库 | 智能物料拣选、库存管理 |
